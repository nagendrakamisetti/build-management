ListDep

Description

Constructs a list of resources and the classes that depend on them. This task is primarily used to determine which Jars are required by the specified list of classes. The task parses each Java file specified in the fileset. It then attempts to load any classes referenced by import statements. The import classes are then added to a hashtable of required resources. Once all classes have been processed, a summary of the required resources will be generated by the task.

Parameters

Attribute Description Required
classpathref Reference to a classpath ID to use when looking up a resource, given as reference to a <path> element defined elsewhere No
reportformat Output format of the dependency summary. Valid formats include text and html. The default format is text. No
reportfile Output file where the summary will be written. No
restrictToJars Determine whether the dependency summary should be restricted to only Jar files. No
displayProgress Enable the display of progress indicators during source code processing. No
includes Comma-separated list of packages specs (may include wildcards). No
excludes Comma-separated list of packages specs (may include wildcards). No

Parameters specified as nested elements

Fileset

Filesets can be used to specify a set of classes to be checked for dependencies. See the Ant FileSet documentation for additional information about how to use the fileset tags.

Examples

List class dependencies found in Java source files

  <listdeps classpathref="ant.classpath" 
               reportformat="html" 
               reportfile="report.html">
    <include name="com.modeln.*"/>
    <exclude name="java.*"/>
    <fileset dir="..">
      <include name="**/*.java"/>
    </fileset>
  </listdeps>
